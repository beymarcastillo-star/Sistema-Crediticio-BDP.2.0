# 5. Alcance del Proyecto (El "Qué")

## 5.1. Funcionalidades Principales

El sistema propuesto contempla el desarrollo de una plataforma virtual que permita digitalizar y optimizar el proceso crediticio para Pyme y Banca Empresa del BDP S.A.M., incorporando las siguientes funcionalidades principales:

- Registro inicial único de datos que alimente automáticamente múltiples formularios del proceso crediticio.
- Implementación de checklists inteligentes que bloqueen el avance del proceso en caso de que falte documentación obligatoria.
- Integración de calculadoras automatizadas para el análisis de flujos operativos y proyecciones agrícolas/pecuarias.
- Generación automática de reportes ejecutivos, incluyendo el "Resumen comité" para la toma de decisiones.
- Selector de perfil de evaluación que permita mostrar u ocultar campos dinámicamente según el tipo de cliente o sector productivo.


## 5.2. Restricciones del Alcance

- No se desarrollará una aplicación móvil nativa independiente; la solución estará limitada a una plataforma web responsiva.
- El sistema dependerá de la correcta integración con el CORE bancario mediante servicios API REST.
- El alcance del proyecto está enfocado exclusivamente al proceso crediticio de Pyme y Banca Empresa, sin incluir otros productos o servicios financieros del banco.
